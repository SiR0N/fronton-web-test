## 📄 Descripción del proyecto

Este proyecto presenta una página web desarrollada con **HTML y CSS básico**. Su objetivo es ofrecer una explicación clara y accesible sobre el deporte del **frontón**, utilizando distintos niveles de títulos, párrafos y varios tipos de listas, incluidas **listas anidadas** para estructurar mejor la información.

En el apartado visual, la página aplica estilos mediante **CSS**, tanto desde la etiqueta `<style>` como a través de **estilos en línea**, modificando colores, fondos y diferentes aspectos del formato del texto.

Además, el proyecto incorpora **imágenes con sus atributos correspondientes** y una organización del contenido que facilita la lectura. En conjunto, el objetivo es practicar:

- La estructura semántica de una página web  
- La organización del contenido  
- La aplicación de estilos básicos para mejorar la presentación
